HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

CSS

  • CSS is used to style HTML.
  • CSS is linked to through a link in the HEAD section.
  • The CSS stylesheet should be save in the same directory as the HTML document.
  • A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color. The X offset is 5px and defines the width or extent of the shadow on the X axis (or to the right of the box); the Y offset is 10px and defines the extent on the Y axis (or just below the box).

GIT

  • -> Git checkout Main (switch back to main to pull it back in after the feature branch has been merged on GitHub)
  • -> Git pull origin main (pull back in the most recent version of main).
