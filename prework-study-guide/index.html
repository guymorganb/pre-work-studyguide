HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is used to style HTML.
  • CSS is linked to through a link in the HEAD section.
  • The CSS stylesheet should be save in the same directory as the HTML document.

GIT

  • Git workflow commands in terminal = Git Status (Check to make sure Im not on the main branch, that Im actuall on the feature branch)
  • -> Git add -A
  • -> Git commit -m "description of commited changes"
  • -> Git pull origin main (pull in the latest version of main)
  • -> Git push origin < name of feature branch to be pushed to Github >
  • -> Git checkout Main (switch back to main to pull it back in after the feature branch has been merged on GitHub)
  • -> Git pull origin main (pull back in the most recent version of main).
